Main

Description

O objetivo desta tarefa é a implementação propriamente funcional do jogo. Embora nas outras tarefas já tenham sido desenvolvidas soluções par aa reação a comandos e passagem do tempo, continua a ser necessária a utilização de uma plataforma de desenvolvimento de uma interface gráfica, reproduzida a partir de cada estado do jogo. Compilada esta tarefa, deve resultar o produto final do projeto.
