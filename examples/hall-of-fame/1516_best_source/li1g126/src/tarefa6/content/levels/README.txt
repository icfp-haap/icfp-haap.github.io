Instructions to add a new level:

1. Just add the map you want with the name:
        "tab$levelnumber$.in"
        example: “tab3.in”

2.  Level will be autamically added to the game
    the next time you start it.

3.  To enable a bomb in a specific level just create a file named:
        bombs$levelnumber$.bomb
        example: "bombs3.bomb"
