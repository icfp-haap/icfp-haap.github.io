Description

Módulo para uma jogada num tabuleiro de Sokoban. Este módulo recebe todas as coordenadas, o tabuleiro em si e uma lista de comandos referentes à movimentação do Sokoban, caso estas movimentações conduzam à finalização do jogo é exibido o FIM juntamente com o número de movimentos, caso os comandos não sejam suficientes para terminar o jogo é exibido INCOMPLETO com o número de movimentos.
