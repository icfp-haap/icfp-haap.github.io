Description

Módulo adapatado da tarefa 4 e que funciona como esqueleto do jogo Sokoban. Este módulo diz respeito à movimentação do Sokoban. Após receber o Input e os comandos necessários à movimentação do Sokoban, a função move fornece um tabuleiro com a posição final do Sokoban após essa movimentação.

Remoção de '#' "redundantes"

Marcação de redundantes

transformCar :: Tabuleiro -> Posicao -> Tabuleiro

Depois de verificados se redundantes, os cardinais que o sejam, são transformados em '@' para serem posteriormente transformados em espaços. Os que não forem redundantes continuam inalterados.

Exemplos de utilização

>>> transformCar ["########","####   #","####   #","########"] (0,0)
["@@@#####","@@@#   #","@@@#   #","@@@#####"]

Verifica redundância

checkCar :: Tabuleiro -> Posicao -> Bool

Verifica se o cardinal é redundante ou não, ou seja, verifica se a toda à volta existem apenas cardinais ou arrobas (@ que correspondem a cardinais já marcados como redundantes).

Exemplos de utilização

>>> checkCar ["@@@#####","@@##   #","####   #","########"] ( 1 , 2 )
True
