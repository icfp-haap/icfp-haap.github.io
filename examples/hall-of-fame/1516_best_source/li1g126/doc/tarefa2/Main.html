Description

Módulo de "vizualização" dos tabuleiros de Sokoban. Este módulo faz a remoção dos caracteres '#' redundantes e faz a coloção do boneco e das caixas no tabuleiro. Retorna o tabuleiro após estas alterações realizadas.

Remoção de '#' "redundantes"

Verifica redundância

checkCar :: Tabuleiro -> Posicao -> Bool

Verifica se o cardinal é redundante ou não, ou seja, verifica se a toda à volta existem apenas cardinais ou arrobas (@ que correspondem a cardinais já marcados como redundantes).

Exemplos de utilização

>>> checkCar ["@@@#####","@@##   #","####   #","########"] ( 1 , 2 )
True

Marcação de redundantes

transformCar :: Tabuleiro -> Posicao -> Tabuleiro

Depois de verificados se são redundantes, os cardinais são transformados em '@' para serem posteriormente transformados em espaços. Os que não forem redundantes continuam inalterados.

Exemplos de utilização

>>> transformCar ["########","####   #","####   #","########"] (0,0)
["@@@#####","@@@#   #","@@@#   #","@@@#####"]

Redundantes para espaço

rmCar :: Tabuleiro -> Tabuleiro

Após a validação, os '#' redundantes (transformados em '@') são substituídos por espaços.

Exemplos de utilização

>>> rmCar ["@@@#####","@@@#   #","@@@#   #","@@@#####"]
["   #####","   #   #","   #   #","   #####"]

Colocação das coordenadas

Colocação geral

colocElem :: Tabuleiro -> [Posicao] -> Int -> Tabuleiro

Função que marca no tabuleiro as posições do boneco e das caixas. Caso seja a primeira vez a correr a função, a posição do boneco corresponde ao 'o' , senão é verificada se a posição é um ponto. Neste caso é colocado um 'I' correspondente à posição final de uma caixa, ou um 'H' (caixa em posição não-final).

Exemplos de utilização

>>> colocElem ["########","####  .#","####  .#","########"] [(4,1),(4,2),(6,2)] 1
["########","####H I#","####o .#","########"]

Troca por Letra

colocAux :: Tabuleiro -> Posicao -> Char -> Tabuleiro

Efetua-se a troca de um lugar (coordenada) pela letra correspondente mencionada anteriormente.

Exemplos de utilização

>>> colocAux ["########","####  .#","####o .#","########"] ( 4 , 2 ) 'H'
["########","####H .#","####o .#","########"]
