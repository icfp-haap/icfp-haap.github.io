Description

Módulo que permite a resolução de um jogo Sokoban. Este módulo recebe um input e certos comandos. O input é o mapa em si com a posição do boneco e as posições das caixas, e de seguida uma das funções gera comandos por ordem crescente de tamanho e uma outra testa se estes mesmos comandos estão corretos. No caso de um estar correto é apresentado como solução.
